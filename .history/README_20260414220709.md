# CompilerX — AetherScript Compiler

A modern, interactive compiler and interpreter built with **Flex**, **Bison**, **React**, and **FastAPI**.

## Features

- **Real-time Lexing & Parsing** — See tokens and AST as you type
- **Interactive AST Visualization** — Zoom, pan, and explore the Abstract Syntax Tree
- **State Inspector** — Watch variable states update during execution
- **C-style Syntax Support** — Variables, functions, loops, conditionals, printf, and more
- **Python Interpreter Backend** — Executes the parsed AST with full output

## Tech Stack

| Layer       | Technology           |
| ----------- | -------------------- |
| Lexer       | Flex (Lex)           |
| Parser      | Bison (Yacc)         |
| Interpreter | Python + FastAPI     |
| Frontend    | React + Vite + D3.js |

## Project Structure

```
compilerX--1/
├── compiler/          # Flex & Bison compiler
│   ├── lexer.l        # Lexical analyzer
│   ├── parser.y       # Bison grammar
│   ├── ast.c / ast.h  # AST node definitions
│   └── main.c         # Compiler driver
├── backend/           # FastAPI interpreter server
│   └── main.py        # AST interpreter & API
└── frontend-react/    # React frontend
    └── src/
        ├── components/
        │   ├── ASTPanel.jsx      # D3.js tree visualization
        │   ├── EditorPanel.jsx   # Code editor
        │   ├── OutputPanel.jsx   # Execution output
        │   └── TokenPanel.jsx    # Token stream display
        └── App.jsx
```

## Quick Start

### 1. Build the Compiler

```bash
cd compiler
make clean && make
```

### 2. Start the Backend Server

```bash
cd backend
../venv/bin/python3 main.py
```

### 3. Start the Frontend

```bash
cd frontend-react
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Supported Syntax

### Variables

```c
int x = 10;
float y = 3.14;
int a = 5, b = 10, temp;
```

### Functions

```c
fn add(int a, int b) {
    return a + b;
}
```

### Control Flow

```c
if (x > 0) {
    print(x);
} else {
    print(-x);
}

while (x > 0) {
    x = x - 1;
}

for (int i = 0; i < 10; i = i + 1) {
    print(i);
}
```

### Printf

```c
printf("a = %d, b = %d\n", a, b);
```

## Example Program

```c
int main() {
    int a = 5, b = 10, temp;

    temp = a;
    a = b;
    b = temp;

    printf("a = %d, b = %d\n", a, b);
    return 0;
}
```

## How It Works

1. **Lexical Analysis** — `lexer.l` (Flex) tokenizes source code
2. **Parsing** — `parser.y` (Bison) builds an AST from tokens
3. **JSON Output** — AST is serialized to JSON
4. **Interpretation** — `main.py` (FastAPI) executes the AST and returns output
5. **Visualization** — React frontend displays tokens, AST, and results
